How to Play Ride the Bus

Overview

Ride the Bus is a card game for 2-10 players with three phases. Players make guesses about cards and face penalties for wrong answers.

Phase One: The Deal (R1-R4)

R1: Guess "Red or Black" - penalty: {{ config.penalty.sips_wrong_guess_r1 if config else 1 }} {{ 'sip' if config and config.alcohol_mode.enabled else 'point' }}
R2: Guess "Higher or Lower" than your first card - penalty: {{ config.penalty.sips_wrong_guess_r2 if config else 1 }} {{ 'sip' if config and config.alcohol_mode.enabled else 'point' }}
R3: Guess "Inside or Outside" the range of your first two cards - penalty: {{ config.penalty.sips_wrong_guess_r3 if config else 1 }} {{ 'sip' if config and config.alcohol_mode.enabled else 'point' }}
R4: Guess the suit - penalty: {{ config.penalty.sips_wrong_guess_r4 if config else 1 }} {{ 'sip' if config and config.alcohol_mode.enabled else 'point' }}, reward: {{ config.reward.reward_distribute_drinks if config else 5 }} {{ 'sips' if config and config.alcohol_mode.enabled else 'points' }} to assign

Phase Two: The Pyramid

Cards are laid in a pyramid (5-4-3-2-1). Cards are flipped one by one, bottom to top. Players with matching ranks can play a card to assign drinks equal to the row value (1-5).

Phase Three: Ride the Bus

The player with the most cards left "rides the bus." 10 cards are flipped. Face cards and Aces give the rider drinks: J=1, Q=2, K=3, A=4.

Special Rules

  • Equal cards in R2/R3 trigger a reshuffle
  • Ties for bus rider are broken by highest remaining card
  • Ace is high in all comparisons
